Compilation
Pour compiler le programme, se mettre à la racine du dossier, puis : 
> cd bin
> javac -cp ../src/minmakespan/minmakespan.java ../src/utils/*.java ../src/minmakespan/*.java

Exécution

Pour exécuter le programme (toujours dans le dossier « bin »), la commande standard est :
> java minmakespan.minmakespan

Plusieurs options sont disponibles en ajoutant un mot clé avant le nom du fichier. La liste des options est disponible avec la commande :
> java -help minmakespan
